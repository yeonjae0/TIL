## Markdown

마크다운의 활용법을 알아보자

1. '#'은 제목을 나타내며 1~6개까지 앞에 붙일 수 있다

2. ```python
   코드를 쓸 수 있다
   ```

3. **볼드**  _이탤릭_  ~~취소선~~     (섞어서도 사용이 가능하다)

4. | 인용도 가능하다

5. -이나 _을 3번 연속하여 선을 그을 수 있다.

6. 링크 [string](url)

7. 이미지 링크 ! [string](url)

8. 표 만들기
   
   제목|내용|설명|
   |---|---|---|
   |테스트1|테스트2|테스트3|
   |테스트1|테스트2|테스트3|
   |테스트1|테스트2|테스트3|
   
   9. 순서와 리스트 (-와 숫자, tab 사용)
      
      1. 순서1
         
         1. 순서2
            
            1. 순서3
      - 리스트
        
        - 리스트2
          
          - 리스트3

## 리눅스

```
파일 이름 찾기 : find . -name file_name  (*를 붙여서 해당 문자 포함 파일 찾기)
한 경로 뒤로 : cd ..
두 경로 뒤로 : cd .. / cd ..
현재 위치의 경로 : pwd 
파일 만들기: cd file_name
내용 지우기 : rm
디렉터리 삭제: rm -r  (디렉터리는 -r 옵션없이 사용 불가)
현재 위치의 파일 목록 조회: ls
디렉터리 생성: mkdir
0바이트 파일 생성: touch
```
